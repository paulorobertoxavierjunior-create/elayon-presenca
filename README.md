# elayon-presenca
Elayon Presença  Conferência inicial de presença humana por fala guiada, com captação temporal, heurística local e geração opcional de relatório avançado.  Objetivo Permitir que o usuário se alinhe consigo mesmo antes de interagir com sistemas maiores, usando a fala guiada, estado anímico em leitura heurística e relatório opcional.
